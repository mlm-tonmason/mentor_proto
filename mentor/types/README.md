# Общие Типы (Types)

Эта директория содержит общие определения моделей данных (Proto Messages & Enums), которые используются во всех слоях архитектуры (Client API, Admin API, Internal Services).

## Содержание

| Файл | Описание |
|------|----------|
| **[common.proto](./common.proto)** | Базовые примитивы: Деньги, Пагинация, Сортировка. |
| **[identity.proto](./identity.proto)** | Пользователи, Профили, Валюты, Настройки. |
| **[bot.proto](./bot.proto)** | Модели Ботов (Ассистентов), Рейтинги, Голоса. |
| **[chat.proto](./chat.proto)** | Чаты, Сообщения, Вложения (Аудио/Видео). |
| **[billing.proto](./billing.proto)** | Тарифы, Подписки, Транзакции, Квоты. |
| **[memory.proto](./memory.proto)** | Память Бота о пользователе (RAG), Саммари. |
| **[call.proto](./call.proto)** | Сессии аудио-звонков, Транскрипты. |

## Принципы

1.  **Переиспользование**: Типы из этой папки импортируются в `mentor/client/` и `mentor/admin/`.
2.  **Язык**: Все комментарии в коде строго на **Русском**.
3.  **Стиль**: Google Protobuf Style Guide (snake_case для полей, CamelCase для сообщений).
