# Mentor Proto

## Описание
**Mentor Proto** — это библиотека определений Protobuf для API сервиса, который предоставляет пользователям возможность взаимодействовать с AI-аватарами.

## Основной функционал
- **Выбор аватара**: Пользователи могут выбирать из списка предварительно обученных аватаров.
- **Общение**: API обеспечивает возможности чата и взаимодействия с выбранным аватаром.

## Планы на будущее (Roadmap)
- **Создание аватаров**: Функционал для создания пользователями собственных аватаров.
- **Обучение**: Инструменты для дообучения аватаров пользователями.
- **Маркетплейс (Аренда)**: Возможность сдавать своих обученных аватаров в аренду другим пользователям.

## Структура
- `mentor/types` — Общие модели данных.
- `mentor/client` — Клиентские сервисы API (RPC).
- `mentor/admin` — Административные сервисы API.
