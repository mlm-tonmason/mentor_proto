# Спецификация UI/UX для Верстки

Этот раздел предназначен для фронтенд-разработчиков (Верстальщиков).
Он связывает визуальные макеты (скриншоты) с бэкенд-логикой (Proto-модели и RPC-методы).


## Структура Экранов

### 1. Авторизация и Профиль
* **[Вход и Профиль](./auth_profile.md)**:
    *   Вход через Telegram Widget.
    *   Личный кабинет (Аватар, Username).
    *   Настройки (Язык, Тема, Анимации).

### 2. Финансы и Подписки
* **[Кошелек и История](./wallet.md)**:
    *   Текущий баланс (TON/USDT).
    *   История транзакций (Пополнения, Списания).
* **[Управление Подписками](./billing.md)**:
    *   Список активных подписок.
    *   Смена тарифа (Upgrade/Downgrade).
    *   Автопродление.

### 3. Маркетплейс (Каталог)
* **[Каталог Ботов](./marketplace.md)**:
    *   Табы: Двойники (Twins) / Эксперты (Experts).
    *   Фильтры (Категории, Hot, Top Rated).
    *   Карточка Бота (Рейтинг, Цена).

### 4. Чат и Библиотека
* **[Библиотека и Активный Чат](./library_chat.md)**:
    *   Сайдбар: Список диалогов, Pinning, Drag & Drop.
    *   Чат: Сообщения (Текст/Голос), История.
    *   Функции: Избранные сообщения, Режим ответа (Text/Voice Widget).
    *   Меню: Переименовать, Архивировать, Удалить.

### 5. Инфо
* **[Поддержка](./support.md)**:
    *   Контакты (Email, Telegram).
    *   Terms & Privacy.
